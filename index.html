Follow the setup steps below to get this URL. Once saved, all submissions flow directly into your Excel file on SharePoint / OneDrive.
One-time setup — connect to Microsoft Excel (5 minutes)
  1. Go to flow.microsoft.com → Create → Instant cloud flow
  2. Choose trigger: "When an HTTP request is received" → Create
  3. Set Method = POST. Copy the HTTP POST URL and paste above.
  4. Add step: Excel Online (Business) → "Add a row into a table"
  5. Create SafeEyes_Log.xlsx on OneDrive with a Table with headers:
    Reference | Date | Time | Name | Phone | Department | Process Area | Area | Zone | Category | Sub-categories | Observation
  6. Map each column to the matching HTTP body field. Save and turn on the flow.
